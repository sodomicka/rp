# Conventions - Elden Ring

- version : W1

## Hierarchie des sources (locale, validee worldbuilder)
1. Decision worldbuilder (tranchages [CANON LOCAL], divergences [DIVERGENCE RP]).
2. BIBLE_LORE + WIKI (lore etabli de cet univers).
3. Canon de 1e main : texte in-game, dialogues, visuel in-game, trailers officiels.
4. Canon secondaire : interviews, artbooks (parole individuelle, premiers jets revises).
5. [DATAMINE] : fichiers du jeu, hors canon explicite.
6. Nightreign : [CANON BIS], univers parallele, jamais fusionne par defaut.
Theories de fans : exclues. Promotion possible uniquement via [CANON LOCAL].
Analystes commu (Vaatividya, Zullie the Witch, Tarnished Archeologist) : sources de FAITS verifies, jamais d'interpretations.

## Balises de certitude
Rappel SPEC : etabli (defaut) / [IMPLICITE] / [INTERPRETATION] / [INCERTAIN] / [DIVERGENCE RP].
Extensions locales :
- [VISUEL] : fait porte par l'image canon seule (jeu ou trailer). L'existence de l'element est canon de 1e main. Sa LECTURE reste a baliser ([IMPLICITE], [INTERPRETATION]) ou a trancher ([CANON LOCAL]).
- [DATAMINE] : fait verifie dans les fichiers du jeu, absent du canon explicite. Avertissement permanent : possible residu d'une decision de lore abandonnee. Verdict worldbuilder par entree : canonise / residu / en attente. Registre : Datamine/Registre.md.
- [CANON LOCAL] : tranchage worldbuilder sur zone muette ou ambigue du canon. Fait foi dans cet univers. Distinct de [DIVERGENCE RP], qui contredit un canon explicite.
- [CANON BIS] : fait issu de Nightreign. Indice possible, verite promue seulement par decision worldbuilder.
