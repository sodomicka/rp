# Outer Gods

- version : W1

## Definition
- Entites exterieures au monde, influencant les Lands Between par receptacles, envoyes, fleaux ou benedictions [etabli].
- Un dieu exterieur peut s'ancrer via un receptacle vivant (deesse, porteur maudit) [etabli ; mecanique detaillee : cf. Cosmogonie/Succession_Divine.md].
