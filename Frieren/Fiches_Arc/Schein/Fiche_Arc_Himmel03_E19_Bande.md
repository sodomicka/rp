## Lore d'arc condense (mini-bible)

- Contexte de l'etape : contrat annexe. Himmel accepte tout (trait canon - il aide tout le monde, missions annexes standard). Le contrat lie Graf Granat a Offen via une prime. Resonance pour Schein : les contrats de chasse sont son mode de vie pendant son ere solitaire (~990-590 av. An 0, ~400 ans de guerilla contre les demons). Se retrouver dans la position de chasseur sous contrat, avec un groupe, est une variation sur ce qu'il a fait seul.
- Lore pertinent :
  - Era solitaire de Schein (~990-590 av. An 0) : guerilla d'usure contre les demons, pas pour les humains mais par necessite de ne pas laisser le Roi renforcer ses troupes. Reputation construite sur ces chasses : le "demon tueur de demons", le "Demon de Lumiere". Les contrats de chasse en groupe sont differents - pas solitaires, pas motives par la survie, mais structurels dans la dynamique de Himmel.
  - Graf Granat comme reseau : le domaine a deja ete traverse a l'Etape 12 (escarmouche avec Aura). Le Graf est en guerre prolongee avec Aura. Ces contrats annexes s'inscrivent dans ce contexte.
- Enjeux et fils chauds :
  - Tchekhov "Empathie de Schein" - mineur. Le retour au role de chasseur - mais en groupe, pour des humains, contre une cible contractuelle. Contraste avec le solitaire d'avant.
