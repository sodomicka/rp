# Kido_et_Disciplines

- version : W1

## Zankensoki (les quatre arts du shinigami)
- Zanjutsu : maniement du sabre. Discipline reine du Gotei (sommet : 11e division).
- Hakuda : combat a mains nues (sommet : Onmitsukido ; Yoruichi Shihoin).
- Hoho : arts du deplacement ; coeur : shunpo (pas eclair). Maitre de reference : Yoruichi, "Deesse du Flash".
- Kido : arts demoniaques. Hado (destruction) et Bakudo (entrave), numerotes 1 a 99 ; kaido (soin) ; kekkai (barrieres).

## Regles du kido
- Incantation complete = pleine puissance. Eishohaki (abandon d'incantation) : plus rapide, puissance reduite, reserve aux experts. Kojutsu eisho : incantation prononcee apres le lancer pour re-amplifier.
- Sorts 90+ : execution extreme, meme pour un capitaine.
- Reperes canon : Hado 31 Shakkaho ; Hado 88 Hiryu Gekizoku Shinten Raiho ; Hado 90 Kurohitsugi ; Bakudo 61 Rikujokoro ; Bakudo 81 Danku (mur bloquant les hado <= 89) ; Bakudo 99 Kin / Bankin.
- Kido Corps : corps autonome specialiste (cf. Factions/Noblesse_et_Institutions.md).

## Techniques transverses
- Shunko : fusion hakuda + kido concentree aux epaules et au dos (Yoruichi, Soifon).
- Kaido de haut niveau : regeneration acceleree (Unohana, Isane, Hachigen pour les barrieres).
- Kekkai de grande echelle : dissimulation, confinement, substitution de zones entieres (Tenkai Kecchu d'Urahara, ere canon).

## Notes
- Les shinigami polyvalents dans les quatre arts sont rares ; la plupart des grades ont une dominante nette.
